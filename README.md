# Presentazione del Javadoc - City Simulator
In questa repository è presente la documentazione Javadoc del progetto City Simulator 
del gruppo ssa_project.
La documentazione JavaDoc del progetto è stata generata automaticamente a partire dai 
commenti presenti nel codice sorgente. Essa fornisce una descrizione delle classi, dei 
metodi e dei relativi parametri, facilitando la comprensione della struttura e delle 
funzionalità del sistema.
